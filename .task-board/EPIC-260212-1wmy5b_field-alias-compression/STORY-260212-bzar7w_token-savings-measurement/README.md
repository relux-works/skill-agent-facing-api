# STORY-260212-bzar7w: token-savings-measurement

## Description
Measure actual token savings from field name abbreviation. Build synthetic benchmarks: take real compact output with full field names (id,name,status,assignee,description) vs abbreviated (i,n,s,a,d). Measure across different payload sizes (5, 20, 100, 500 items). Compare: full-name compact vs abbreviated compact vs JSON baseline. Calculate % savings per abbreviation strategy. Use tiktoken or Claude tokenizer for accurate counts.
