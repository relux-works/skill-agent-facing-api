# STORY-260213-kgw7l0: list-pagination-skip-take

## Description
Add skip/take pagination support for list() operations. When a list() returns many items, the caller should be able to paginate with skip(N) and take(M) parameters. Standard approach: skip=offset, take=limit. Must work with field projection and compact output format.
